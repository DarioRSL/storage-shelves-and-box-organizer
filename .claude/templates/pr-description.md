## Usage in Skill

The `/commit-pr-issue` skill will:

1. Gather PR data (commits, diff stats, files)
2. Analyze commits to extract content
3. Generate each section using templates:
   - Summary: Synthesize from commit messages
   - Key Features: Extract highlights
   - Changes Overview: Group commits by category
   - Technical: Extract architecture details from commit bodies
   - Testing: Use template or user-provided results
   - Files: Count and categorize from git diff
   - Deployment: Generate type-specific checklist
   - Issues: Insert from issue search phase
   - Next Steps: Suggest based on incomplete work or future enhancements
   - Documentation: List .ai_docs changes
   - Checklist: Verify against requirements
4. Calculate footer metadata (commit count, dates, stats)
5. Apply template with all variables filled
